# Research

## Root cause

1. `selectCandidates` sorts the same candidate pool by CAPEX for all policies, so the same lowest-scored template wins independently.
2. `SEED_PRICE_QUOTES` is empty by design, and `formatPrice` hides all numeric values whenever `quotationRequired` is true.
3. The report BOM maps specifications only and has no component-cost model.
