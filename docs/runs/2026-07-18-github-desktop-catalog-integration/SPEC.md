# Specification — GitHub desktop catalog integration

- Preserve Windows 11 x64, macOS 26 arm64 and Ubuntu 24.04 x64 desktop support.
- Adopt the 14-template catalog, explicit target OS, exact existing hardware, conservative integrated/mobile sizing and differentiated proposals.
- Add dated componentized reference costs while keeping seller quotes authoritative and purchase quotation mandatory for reference estimates.
- Export JSON schema `capacity-recommendation-export/2.2.0` and numeric, reconciled costs in PDF/XLSX/JSON.
- Default legacy scenarios to target OS `auto` and no exact template.
- Preserve the current database schema, filename, `userData` resolution and every desktop security/lifecycle invariant.
- Ship no standalone server, Docker runtime or compiled server/worker entry point in ASAR.
