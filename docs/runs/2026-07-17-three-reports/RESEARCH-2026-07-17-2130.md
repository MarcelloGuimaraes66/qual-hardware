# Research

- The calculation already creates three `CapacityRecommendation` records per revision: `minimum`, `recommended`, and `n_plus_one`.
- The export endpoint currently selects one recommendation by id and passes only that object to all report writers.
- The UI currently hides the catalog button unless both remote URL and public key came from process environment variables.
- The signed snapshot parser and Ed25519 verification already exist, so configuration and manual import can reuse the same trust boundary.
- PDF is generated with `pdf-lib`; XLSX is generated with the existing runtime dependency ExcelJS and will be independently inspected with the bundled spreadsheet tooling.
