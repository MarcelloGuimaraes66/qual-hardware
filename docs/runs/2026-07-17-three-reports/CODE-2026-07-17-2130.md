# Code

## Implemented

- Consolidated PDF, XLSX and JSON exports now resolve the complete ordered recommendation set: minimum, recommended and N+1.
- PDF has a comparison page and a detailed, visually separated section for each configuration.
- XLSX has policy-aware summary, BOM, node, workload, calculation, quote and assumption sheets.
- The desktop footer always displays **Atualizar hardware**.
- The catalog manager persists its HTTPS source and Ed25519 public verification key in the desktop user-data directory.
- A signed JSON catalog can be selected locally and imported through the same validation path used by online updates.
- Invalid, unsigned or incompatible catalogs do not replace the active catalog.
