# Planning

- Project: standalone Aiquimist **Qual Hardware**, repository root `qual-hardware`.
- Process authority: global Archon pointer; no repository-local `.archon` memory is available.
- Risk: T3. The change crosses API, PDF/XLSX/JSON generation, desktop catalog configuration, UI and security validation.
- Change budget: targeted redesign limited to consolidated reports and signed catalog management.
- Excluded: capacity calculations, Perceptrum runtime, SQLite scenario schema, deployment and benchmark rules.
- Rollback: restore the single-recommendation report context and environment-only catalog configuration.

## Invariants

- Every export contains exactly the current minimum, recommended and N+1 primary designs.
- A catalog is never applied without an Ed25519 signature verified by the configured public key.
- Invalid imports/configuration leave the last valid catalog active.
- Reports contain no media or RTSP credentials and existing scenario data remains readable.
- Windows desktop remains the validated packaging target; web/API logic stays platform-neutral.
