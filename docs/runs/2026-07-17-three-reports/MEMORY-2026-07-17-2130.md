# Memory

- There is no repository-local Archon memory store, so durable decisions were recorded in `README.md`, `docs/CATALOG_UPDATES.md`, `docs/VALIDATION.md` and this run directory.
- Reports are a three-configuration contract. A partial recommendation set is an export error, not a valid reduced report.
- Hardware catalog trust is anchored by a user-configured Ed25519 public key. Manual import is not an unsigned escape hatch.
- Catalog configuration is writable by the desktop application and opt-in for standalone server deployments.
- Sample report artifacts are generated by `npm run reports:sample` and remain outside version control.
