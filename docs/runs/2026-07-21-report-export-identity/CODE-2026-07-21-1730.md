# Código - identidade dos downloads

- `src/web/reportExports.ts`: fonte única para formatos, nomes e textos da exportação.
- `src/web/App.tsx`: usa os nomes canônicos, apresenta o relatório principal em destaque e recolhe o anexo neutro.
- `src/web/styles.css`: separação visual e responsiva entre os dois documentos.
- `tests/report-export-identity.test.ts`: trava nomes e mensagens para impedir nova ambiguidade.
