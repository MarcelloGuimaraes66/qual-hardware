# Código - composição justificada

- Quebra de palavras pela largura real retornada por `PDFFont.widthOfTextAtSize`.
- Desenho palavra a palavra para ocupar a largura útil.
- Limite de 2,2 vezes o espaço natural para evitar rios tipográficos.
- Método `paragraph` com suporte a texto regular ou em negrito.
- Chamadas seletivas apenas nos blocos definidos na especificação.
- Nova página antes de “Outras máquinas qualificadas” para preservar a hierarquia visual após a mudança de paginação.
