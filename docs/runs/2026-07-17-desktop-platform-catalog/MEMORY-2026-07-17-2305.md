# Durable decisions

- “Can run Perceptrum” and “validated camera capacity” are separate claims. User observation admits a template to estimated planning; only the sustained benchmark validates it.
- Core count alone cannot size mobile heterogeneous CPUs. Until calibrated, mobile templates require a visible conservative derating and explicit media-preparation limits.
- NPU TOPS are not credited to local AiQ unless the Perceptrum runtime actually dispatches that workload to the NPU and a matching benchmark proves it.
- Apple unified memory is never reported as dedicated VRAM. Apple designs are opt-in and require a macOS/Apple Silicon Perceptrum build.
